# イベント申込者一覧の機能

イベントへの申込者情報を確認・管理する機能です。申込状況の確認、検索・絞り込みが行えます。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバーの **「イベント申込者一覧」** をクリック

![イベント申込者一覧画面](/images/organizer/applicant-list.png)

## 申込者の表示項目

一覧テーブルには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **申込み日時** | 申込みが行われた日時。クリックで並び替え可能 |
| **イベント名** | 申込先のイベントの名称 |
| **チケット名** | 参加者が選択したイベントチケットの種類 |
| **会社名** | 申込者の所属する会社名 |
| **担当領域** | 申込者の職種や担当業務 |
| **キャンセル日付** | 参加のキャンセルがあった場合、そのキャンセル処理が行われた日付 |

## 検索・絞り込み機能

以下の条件で申込者を絞り込めます：

| 絞り込み条件 | 説明 |
|-------------|------|
| **申込み期間** | カレンダーから開始日〜終了日の日付範囲で絞り込み |
| **イベントNo.** | イベント管理番号で検索 |
| **イベント名** | イベント名のテキスト検索 |
| **企業名・申込者名** | 申込者の会社名や氏名でキーワード検索 |

::: tip
複数の条件を組み合わせて絞り込むことも可能です。
:::

## 表示件数の変更

一覧の右上にある「表示件数」から、1ページあたりの表示件数を変更できます。

## キャンセル管理

申込者がキャンセルした場合、「キャンセル日付」列にキャンセル日が自動的に表示されます。

::: info
参加者がキャンセルを行うと、自動的にこの一覧に反映されます。主催者側で手動キャンセルを行う必要はありません。
:::
