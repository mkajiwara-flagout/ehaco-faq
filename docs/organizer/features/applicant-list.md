# イベント申し込み者一覧の機能について

イベントごとの申込者情報を確認・管理する機能です。申込日時、イベント名、チケット名、会社名、担当領域、キャンセル状況などを確認でき、期間での絞り込みやCSVデータの出力ができます。

## 申込者の表示項目

| 項目 | 説明 |
|------|------|
| 申込み日時 | 申込みが行われた日時 |
| イベント名 | 申込先のイベント名 |
| チケット名 | 選択されたチケット名 |
| 会社名 | 申込者の会社名 |
| 担当領域 | 申込者の担当領域 |
| キャンセル日付 | キャンセルされた場合の日付 |

表示件数の変更やページネーション機能も利用可能です。

## 検索・絞り込み機能

以下の条件で絞り込みが可能です：

- 申込み期間での絞り込み
- イベント名での検索
- チケット名での検索
- 会社名での検索

## データ管理機能

### CSV出力機能

- 申込者情報の出力
- アンケート回答の出力
- キャンセル情報の出力

### 申込状況の管理

- 参加状況の確認
- キャンセル対応
- ステータス管理
- 参加者数の把握

## キャンセル管理

キャンセルに関する以下の情報を確認できます：

- キャンセル日時
- キャンセル理由
- キャンセル者の情報
