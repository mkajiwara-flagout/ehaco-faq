# イベントページ一覧の機能

作成したイベントの確認・管理を行う機能です。イベントの状態確認や編集、データのエクスポートが行えます。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバーの **「イベントページ一覧」** をクリック

![イベントページ一覧画面](/images/organizer/event-list.png)

## 絞り込み・検索機能

イベントの一覧を以下の条件で絞り込めます：

| 絞り込み条件 | 説明 |
|-------------|------|
| **開催期間** | カレンダーから開催期間の範囲を指定 |
| **イベント名・イベントNo.** | テキスト検索で絞り込み |
| **アンケート名** | 使用しているアンケートで絞り込み |
| **ターゲット名** | 使用しているターゲットで絞り込み |

## ステータスタブ

画面上部のタブでイベントの状態ごとに表示を切り替えられます：

| タブ | 説明 |
|------|------|
| **すべて** | 全イベントを表示 |
| **受付中** | 現在申込みを受け付けているイベント |
| **公開前** | 公開日時前のイベント |
| **下書き** | 作成途中で未公開のイベント |
| **終了** | 募集期間が終了したイベント |

### ステータスの流れ

```
下書き → 公開前 → 受付中 → 終了
```

## 一覧で確認できる情報

各イベントカードには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **イベント画像** | イベントのサムネイル画像 |
| **イベント名** | 作成したイベントのタイトル。クリックするとイベント詳細を表示 |
| **イベントNo.** | イベント固有の管理番号 |
| **募集人数** | 設定した募集定員 |
| **申込人数** | 現時点の申込者数 |
| **応募期限** | 募集の締め切り日時 |

## イベントの操作

各イベントカードから以下の操作が行えます：

| ボタン | 説明 |
|--------|------|
| **イベントの編集** | イベント情報の編集画面に移動します |
| **Excelダウンロード** | 申込者データをExcelファイルとしてダウンロードできます |

::: warning 注意
「受付中」のイベントは、一部の項目のみ編集可能です。募集人数やチケット設定など、申込者に影響する項目は変更できない場合があります。
:::

::: tip
申込者の詳細な管理は [イベント申込者一覧](/organizer/features/applicant-list) から行えます。
:::
