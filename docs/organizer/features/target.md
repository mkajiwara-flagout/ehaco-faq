# ターゲットの機能について

イベントの対象者を設定し、検索されやすくするための機能です。業種、従業員数、年商規模、職種、立場などの条件を設定できます。

## ターゲット作成

### 文字数制限

| 項目 | 制限 |
|------|------|
| ターゲット名 | 25文字以内 |

### 選択項目

| 項目 | 選択方式 |
|------|---------|
| 業種 | 複数選択可能（チェックボックス） |
| 従業員数 | 単一選択（ラジオボタン） |
| 年商規模 | 単一選択（ラジオボタン） |
| 職種 | 複数選択可能（チェックボックス） |
| 立場 | 複数選択可能（チェックボックス） |

## ターゲット管理

- 作成したターゲットの管理
- ターゲット一覧での確認
- 使用状況の確認
- ターゲットの削除（未使用時のみ）

## ターゲットの編集・削除

| 条件 | 編集可否 |
|------|---------|
| イベントで未使用の場合 | 編集可能 |
| ステータスが有効の場合 | 編集可能 |
| イベントで使用中 | 編集不可 |
| イベントで使用実績がある場合 | 編集不可 |
