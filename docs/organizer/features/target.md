# ターゲットの機能について

イベントの対象者を設定し、検索されやすくするための機能です。業種、従業員数、年商規模、職種、立場などの条件を設定できます。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバーの **「ターゲット設定」** をクリック

![ターゲット一覧画面](/images/organizer/target-list.png)

## ターゲット一覧

ターゲット一覧画面では、作成済みのターゲットを管理できます。

### 絞り込み機能

| 条件 | 説明 |
|------|------|
| **作成期間** | カレンダーから日付範囲で絞り込み |
| **ターゲット名** | キーワードで検索 |

### 一覧の表示項目

| 項目 | 説明 |
|------|------|
| **作成日** | ターゲットが作成された日時 |
| **ターゲット名** | ターゲットの識別名称 |
| **使用しているイベント数** | このターゲットを使用しているイベントの数。クリックで該当イベント一覧に移動 |

## ターゲット作成

ターゲット一覧画面の右上にある **「ターゲット作成」** ボタンをクリックして新規作成します。

### 文字数制限

| 項目 | 制限 |
|------|------|
| ターゲット名 | 25文字以内 |

### 選択項目

| 項目 | 選択方式 |
|------|---------|
| **業種** | 複数選択可能（チェックボックス） |
| **従業員数** | 単一選択（ラジオボタン） |
| **年商規模** | 単一選択（ラジオボタン） |
| **職種** | 複数選択可能（チェックボックス） |
| **立場** | 複数選択可能（チェックボックス） |

## ターゲットの編集・削除

| 条件 | 編集可否 |
|------|---------|
| イベントで未使用の場合 | 編集可能 |
| ステータスが有効の場合 | 編集可能 |
| イベントで使用中 | **編集不可** |
| イベントで使用実績がある場合 | **編集不可** |

::: warning 注意
イベントで使用中のターゲットは編集できません。新しいターゲットを作成してください。
:::
