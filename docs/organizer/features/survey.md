# アンケートの機能について

イベント申込時に参加者に回答してもらうアンケートを作成・管理する機能です。

## アンケート作成

### 文字数制限

| 項目 | 制限 |
|------|------|
| アンケート名 | 25文字以内 |
| 設問文 | 50文字以内 |
| 選択肢 | 50文字以内 |

### 設問数

- 最大 **3問** まで設定可能
- すべて必須回答
- 順序の変更可能

### 回答形式

| 形式 | 説明 |
|------|------|
| 単一回答式 | 選択肢から1つのみ選択。ラジオボタンで表示。最大10個まで選択肢作成可能 |
| 複数回答式 | 選択肢から複数選択可能。チェックボックスで表示。最大10個まで選択肢作成可能 |

## アンケート管理

- 作成したアンケートの管理
- アンケート一覧での確認
- 使用状況の確認
- アンケートの削除（未使用時のみ）

## アンケートの編集・削除

| 条件 | 編集可否 |
|------|---------|
| イベントで未使用の場合 | 編集可能 |
| 回答データが存在しない場合 | 編集可能 |
| イベントで使用中 | 編集不可 |
| 回答データが存在する場合 | 編集不可 |
