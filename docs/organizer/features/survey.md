# アンケートの機能について

イベント申込時に参加者に回答してもらうアンケートを作成・管理する機能です。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバーの **「アンケート設定」** をクリック

![アンケート一覧画面](/images/organizer/survey-list.png)

## アンケート一覧

アンケート一覧画面では、作成済みのアンケートを管理できます。

### 絞り込み機能

| 条件 | 説明 |
|------|------|
| **作成期間** | カレンダーから日付範囲で絞り込み |
| **アンケート名** | キーワードで検索 |

### ステータスタブ

| タブ | 説明 |
|------|------|
| **使用可** | 現在使用可能なアンケート |
| **アーカイブ** | アーカイブ済みのアンケート |
| **すべて** | 全アンケートを表示 |

### 一覧の表示項目

| 項目 | 説明 |
|------|------|
| **作成日** | アンケートが最初に作成された日時 |
| **アンケート名** | アンケートの識別名称 |
| **ステータス** | 「使用可」などの現在の状態 |
| **アンケート項目数** | そのアンケートに含まれる質問の数（例：1/3） |
| **使用しているイベント数** | 現在このアンケートを使用しているイベントの数。クリックで該当イベント一覧に移動 |
| **アンケート回答者数** | そのアンケートに回答した参加者の総数。クリックで該当申込者一覧に移動 |

## アンケート作成

アンケート一覧画面の右上にある **「アンケート作成」** ボタンをクリックして新規作成します。

### 文字数制限

| 項目 | 制限 |
|------|------|
| アンケート名 | 25文字以内 |
| 設問文 | 50文字以内 |
| 選択肢 | 50文字以内 |

### 設問数

- 最大 **3問** まで設定可能
- すべて必須回答
- 順序の変更可能

### 回答形式

| 形式 | 説明 |
|------|------|
| **単一回答式** | 選択肢から1つのみ選択。ラジオボタンで表示。最大10個まで選択肢作成可能 |
| **複数回答式** | 選択肢から複数選択可能。チェックボックスで表示。最大10個まで選択肢作成可能 |

## アンケートの編集・削除

| 条件 | 編集可否 |
|------|---------|
| イベントで未使用の場合 | 編集可能 |
| 回答データが存在しない場合 | 編集可能 |
| イベントで使用中 | **編集不可** |
| 回答データが存在する場合 | **編集不可** |

::: warning 注意
使用しているイベントが存在する場合、アンケートの編集はできません。新しいアンケートを作成してください。
:::
