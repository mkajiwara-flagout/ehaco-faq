# 担当者一覧の機能

イベントを運営する担当者（スタッフ）の追加・管理を行う機能です。複数の担当者でイベント運営を行う場合に利用します。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバー下部の **「担当者一覧」** をクリック

![担当者一覧画面](/images/organizer/staff-list.png)

## 一覧で確認できる情報

| 項目 | 説明 |
|------|------|
| **主催担当者ID** | 担当者を一意に識別するための固有の番号 |
| **担当者名** | 登録された担当者の氏名 |
| **メールアドレス** | 担当者が登録しているログイン用メールアドレス |
| **電話番号** | 担当者が登録している電話番号 |
| **状態** | 「登録完了」または「登録待ち」 |

### 検索機能

「担当者名・メールアドレスで絞り込む」テキストボックスで、担当者を検索できます。

## 担当者の追加（招待）

新しい担当者を追加するには、画面右上の **「担当者の追加」** ボタンをクリックします。

1. 追加する担当者の **姓・名**（全角漢字）を入力
2. 担当者の **ログイン用メールアドレス** を入力
3. 招待メールが自動送信されます

### 登録状態について

| 状態 | 説明 |
|------|------|
| **登録完了** | アカウント登録が完了し、管理画面にログイン可能 |
| **登録待ち** | 招待メール送信済みだが、まだ登録が完了していない |

::: tip
招待メールが届かない場合は、迷惑メールフォルダの確認をお願いしてください。
:::

::: warning 注意
招待メールに含まれるURLの有効期限は **1時間30分** です。期限内に必ず登録を完了してください。
:::

## 担当者情報の編集

一覧の各担当者の行にある編集アイコン（鉛筆マーク）をクリックすると、以下の情報を変更できます：

| 編集可能な項目 | 備考 |
|--------------|------|
| **姓** | 全角漢字 |
| **名** | 全角漢字 |
| **電話番号** | - |

::: info
メールアドレスは変更できません。変更が必要な場合は、担当者を削除して再度招待してください。
:::
