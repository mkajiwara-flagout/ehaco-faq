# 担当者一覧の機能について

イベントを運営する担当者の追加・管理を行う機能です。担当者の登録状態の確認や、アカウントの設定変更ができます。

## 一覧での表示項目

| 項目 | 説明 |
|------|------|
| 主催担当者ID | 担当者の識別ID |
| 担当者名 | 登録された担当者の氏名 |
| メールアドレス | ログイン用メールアドレス |
| 電話番号 | 連絡用電話番号 |
| 登録状態 | 登録完了 or 登録待ち |

## 担当者の状態

| 状態 | 説明 |
|------|------|
| 登録完了 | 利用可能な状態 |
| 登録待ち | 招待メール送信済み |

## 設定機能

以下の情報を変更できます：

- 姓（全角漢字）
- 名（全角漢字）
- 電話番号
- 担当者の削除
