# 主催者情報の機能について

主催企業・団体の基本情報や連絡先を設定・管理する機能です。表示名、問い合わせURL、プロフィール画像など、公開情報の設定ができます。

## アクセス方法

1. [管理画面](/organizer/features/dashboard) にログイン
2. 左サイドバー下部の **「主催者情報」** をクリック

![主催者情報設定画面](/images/organizer/organizer-info.png)

## 設定項目一覧

### 基本情報

| 項目 | 説明 | 変更 |
|------|------|------|
| **表示名** | イベント一覧やイベント詳細ページに表示される主催者名。一般公開されます | 可能 |
| **専用ページURL** | `https://ehaco.net/organizer/[任意の文字列]` 形式で発行されるURL | **変更不可** |
| **企業名または団体名** | 正式名称。運営管理用 | 可能 |

::: warning 注意
専用ページURLは後から変更することができません。入力には半角英数字とハイフンのみ利用可能で、既に使用されているURLは利用できません。
:::

### プロフィール設定

#### プロフィール表示画像

| 項目 | 仕様 |
|------|------|
| 推奨サイズ | 240 x 240px |
| 対応形式 | jpg、jpeg、gif、png |
| ファイルサイズ | 最大1MB |

#### 紹介文

企業・団体の説明文を入力できます。一般公開される情報です。

### 連絡先情報

| 項目 | 公開/非公開 | 説明 |
|------|-----------|------|
| **お問い合わせURL** | 公開 | サイト上で一般公開されます。参加者からの問い合わせ用URLです |
| **お問い合わせメールアドレス** | 非公開 | サイト上では公開されません。申込者に対するメールで使用します |
| **企業・団体のWebサイトURL** | 公開 | 企業・団体のホームページURL |
| **企業または団体の電話番号** | 非公開 | 利用者には公開されません。ehaco!運営事務局からの連絡用です |

## 公開される情報・非公開の情報

| 公開される情報 | 非公開の情報 |
|-------------|------------|
| 表示名 | お問い合わせメールアドレス |
| お問い合わせURL | 電話番号 |
| WebサイトURL | |
| プロフィール表示画像 | |
| 紹介文 | |

## 変更の保存

設定を変更したら、画面下部の **「変更を保存」** ボタンをクリックして保存してください。
