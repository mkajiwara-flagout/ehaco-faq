# 主催者情報の機能について

主催企業・団体の基本情報や連絡先を設定・管理する機能です。表示名、問い合わせURL、プロフィール画像など、公開情報の設定ができます。

## 基本設定項目

| 項目 | 説明 | 変更 |
|------|------|------|
| 表示名（必須） | 企業プロフィールの表示名。一般公開される情報 | 可能 |
| 専用ページURL | `https://ehaco.net/organizer/[任意の文字列]` 形式。半角英数字とハイフンのみ使用可能 | 不可 |
| 企業名または団体名 | 正式名称。運営管理用 | 不可 |

## プロフィール設定

### プロフィール画像

| 項目 | 仕様 |
|------|------|
| サイズ | 240 x 240px |
| 対応形式 | jpg、jpeg、gif、png |
| ファイルサイズ | 最大1MB |

### 紹介文

企業・団体の説明文を入力できます。一般公開される情報です。

## 連絡先情報

| 項目 | 公開/非公開 | 説明 |
|------|-----------|------|
| お問い合わせURL（必須） | 公開 | 参加者からの問い合わせ用 |
| お問い合わせメールアドレス（必須） | 非公開 | 申込者とのメールでの連絡用 |
| WebサイトURL（必須） | 公開 | 企業・団体のホームページ |
| 電話番号（必須） | 非公開 | 運営事務局からの連絡用 |

## 公開される情報・非公開の情報

| 公開される情報 | 非公開の情報 |
|-------------|------------|
| 表示名 | お問い合わせメールアドレス |
| お問い合わせURL | 電話番号 |
| WebサイトURL | |
| プロフィール画像 | |
| 紹介文 | |

::: tip URL設定の制限
- 半角英数字のみ
- ハイフン使用可
- スペース使用不可
- 重複不可
:::
