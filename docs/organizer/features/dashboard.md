# 管理画面の概要

主催者としてログインすると、イベントの作成・管理、申込者の確認、各種設定を行える管理画面にアクセスできます。

## 管理画面へのアクセス

1. [org.ehaco.net](https://org.ehaco.net) にアクセス
2. メールアドレスとパスワードでログイン
3. 管理画面が表示されます

## サイドバーメニュー

![イベントページ一覧画面](/images/organizer/event-list.png)

ログイン後、画面左側のサイドバーから各機能にアクセスできます。

| メニュー | 内容 | 詳細ページ |
|----------|------|------------|
| **イベント申込者一覧** | イベントへの申込者の確認・管理 | [イベント申込者一覧の機能](/organizer/features/applicant-list) |
| **イベントページ一覧** | 作成したイベントの確認・管理 | [イベントページ一覧の機能](/organizer/features/event-list) |
| **イベント新規登録** | 新しいイベントの作成 | [イベント作成](/organizer/getting-started/step5-create-event) |
| **アンケート設定** | 事前アンケートの作成・管理 | [アンケートの機能](/organizer/features/survey) |
| **ターゲット設定** | 対象者グループの作成・管理 | [ターゲットの機能](/organizer/features/target) |
| **ヘルプページ** | ヘルプセンターへのリンク | - |

サイドバー下部には以下のメニューもあります：

| メニュー | 内容 | 詳細ページ |
|----------|------|------------|
| **主催者情報** | 主催者プロフィールの編集 | [主催者情報の機能](/organizer/features/organizer-info) |
| **担当者一覧** | 担当者（スタッフ）の管理 | [担当者一覧の機能](/organizer/features/staff-list) |

## イベント運営の基本フロー

管理画面では以下の流れでイベントを運営します：

### 1. 事前準備

| 手順 | 説明 |
|------|------|
| アンケート作成 | 参加者に回答してもらうアンケートを事前に作成 |
| ターゲット設定 | 対象者の条件（業種・規模など）を設定 |

### 2. イベント作成・公開

| 手順 | 説明 |
|------|------|
| イベント新規登録 | イベント情報を入力して作成 |
| 下書き保存 or 公開 | 内容を確認してから公開 |

### 3. 募集期間中

| 手順 | 説明 |
|------|------|
| 申込者確認 | イベント申込者一覧で参加者を確認 |
| Excelエクスポート | 申込者リストをExcelファイルでダウンロード |

### 4. イベント終了後

| 手順 | 説明 |
|------|------|
| アンケート結果確認 | 回答データを確認 |
| 参加者データ管理 | 必要に応じて申込者データをエクスポート |

## ログアウト

管理画面からログアウトするには、画面右上のユーザーアイコンをクリックし、 **「ログアウト」** を選択してください。
