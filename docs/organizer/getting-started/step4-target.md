# ステップ4：ターゲット作成

## 1. ターゲット一覧画面へのアクセス

1. 主催者管理画面にログイン
2. 左側のサイドバーから「ターゲット一覧」をクリック

## 2. ターゲット作成画面を開く

ターゲット一覧画面の右上にある「ターゲット作成」ボタンをクリックします。

## 3. ターゲット情報の入力

以下の情報を入力してください：

| 項目 | 説明 |
|------|------|
| ターゲット名 | ターゲットの名前を入力 |
| 業種 | 対象の業種を選択 |
| 従業員数 | 対象の従業員規模を選択 |
| 年商規模 | 対象の年商規模を選択 |

## 4. ターゲットの保存

すべての必要項目を入力し、「保存」ボタンをクリックすると、作成したターゲットがターゲット一覧に表示されます。

::: tip
ターゲットの詳しい仕様については [ターゲットの機能](/organizer/features/target) をご覧ください。
:::
