# ステップ3：アンケート作成

## 1. アンケート一覧画面へのアクセス

1. 主催者管理画面にログイン
2. 左側のサイドバーから「アンケート設定」をクリック

## 2. アンケート作成画面を開く

アンケート一覧画面の右上にある「アンケート作成」ボタンをクリックします。

## 3. アンケート情報の入力

以下の情報を入力してください：

| 項目 | 説明 |
|------|------|
| アンケート名 | アンケートの名前を入力 |
| 回答形式 | 単一回答式（1つだけ選択）または複数回答式（複数選択可能） |
| 設問文 | 質問の内容を入力 |
| 回答 | 選択肢を入力 |

## 4. アンケートの作成完了

すべての必須項目を入力し、「作成する」ボタンをクリックすると、アンケート一覧に作成したアンケートが表示されます。

::: tip
アンケートの詳しい仕様については [アンケートの機能](/organizer/features/survey) をご覧ください。
:::
