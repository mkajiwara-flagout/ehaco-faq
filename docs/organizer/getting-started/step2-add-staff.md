# ステップ2：担当者追加

## 1. 担当者一覧画面へのアクセス

1. 主催者管理画面にログイン
2. 左側のサイドバー下部から「担当者一覧」をクリック

## 2. 担当者追加画面を開く

担当者一覧画面の右上にある「担当者の追加」ボタンをクリックします。

## 3. 担当者情報の入力

以下の情報を入力してください：

- 姓（全角漢字）
- 名（全角漢字）
- 担当者のログイン用メールアドレス

## 4. 担当者情報の登録後

担当者を追加すると：

- 一覧に新しい担当者が「登録待ち」状態で表示されます
- 登録したメールアドレスに招待メールが自動送信されます

## 5. 招待メールの確認

- メール本文の登録用URLをクリック
- ブラウザで登録画面が開きます

::: warning 注意
URLの有効期限は **1時間30分** です。期限内に必ず登録を完了してください。
:::

## 6. 担当者情報の登録

以下の情報を入力してください：

- 電話番号
- パスワード
- パスワード再入力

入力後「主催企業の担当者情報として登録する」をクリックして完了です。
