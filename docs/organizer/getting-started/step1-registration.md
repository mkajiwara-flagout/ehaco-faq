# ステップ1：新規主催者登録

## 1. 新規登録画面へのアクセス

- ehaco! の主催者ログイン画面にアクセス
- 画面下部の「新規主催者登録はこちら」をクリック

![主催者ログイン画面](/images/organizer/org-login-page.png)

## 2. 企業・団体情報の入力

![主催者登録フォーム](/images/organizer/registration-page.png)

以下の情報を入力してください：

| 項目 | 説明 |
|------|------|
| 表示名 | イベント一覧ページやイベント詳細ページに表示される主催者名です |
| 専用ページURL | 主催者の詳細情報を掲載するページのURLとなります |
| お問い合わせURL | イベント詳細ページに表示される問い合わせ先URLです |
| 企業名または団体名 | 正式な企業名・団体名を入力してください |
| 企業または団体の電話番号 | 運営事務局からの連絡用の電話番号です。参加者には表示されません |
| お問い合わせメールアドレス | |
| 企業・団体のWebサイトURL | イベント詳細ページに表示される主催者のWebサイトURLです |

## 3. 担当者情報の入力

以下の情報を入力してください：

- 担当者名（姓・名）
- 担当者のログイン用メールアドレス
- 担当者のログイン用パスワード
- 担当者の電話番号

## 4. 利用規約の確認

1. 利用規約を確認
2. 同意チェックボックスにチェック
3. 「利用規約に同意して入力情報を送信する」をクリック

## 5. 登録完了

登録申請が完了すると確認画面が表示されます。

## 6. 承認メールの確認

運営事務局での確認後、登録完了メールが届きます。メールが届きましたら、ログインが可能になります。
