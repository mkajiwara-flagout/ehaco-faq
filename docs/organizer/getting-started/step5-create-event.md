# ステップ5：イベント作成

## 1. イベント新規登録画面へのアクセス

1. 主催者管理画面にログイン
2. 左側のサイドバーから「イベント新規登録」をクリック

## 2. 基本情報の入力

以下の情報を入力してください：

- イベント名
- 開催日時
- 募集日時
- イベント画像
- イベント詳細文

## 3. 開催方法の設定

### オフライン開催の場合

| 項目 | 説明 |
|------|------|
| 開催場所名 | 会場名を入力 |
| 開催場所住所 | 会場の住所を入力 |
| 都道府県 | 都道府県を選択 |

### オンライン開催の場合

| 項目 | 説明 |
|------|------|
| 開催ツール名 | 使用するツール名を入力 |
| 視聴用URL | 参加者向けのURLを入力 |

## 4. イベントチケットの入力

| 項目 | 説明 |
|------|------|
| イベントチケット名 | 日程や開催場所を記入（例：「4月15日 東京会場」） |
| 枚数（必須） | 半角数字で入力 |

## 5. アンケート・ターゲットの設定

- アンケートの選択（事前に作成が必要）
- ターゲットの選択（事前に作成が必要）
- 興味タグの設定

::: tip
アンケートとターゲットは事前に作成しておく必要があります。
- [ステップ3：アンケート作成](/organizer/getting-started/step3-survey)
- [ステップ4：ターゲット作成](/organizer/getting-started/step4-target)
:::

## 6. 公開設定とメッセージ

- 公開方法の選択
- 主催者からのメッセージ

## 7. イベントの保存

| 操作 | 説明 |
|------|------|
| 下書き保存 | 後で編集する場合、公開前の確認用 |
| 保存して公開 | すぐに公開する場合、申込受付開始 |
