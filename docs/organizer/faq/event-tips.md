# イベント作成のコツ

より多くの参加者に見つけてもらい、申し込みにつなげるためのイベント作成のポイントをご紹介します。

## イベント名のコツ

イベント名は検索結果やイベント一覧で最初に目に入る要素です。

| ポイント | 良い例 | 改善が必要な例 |
|---------|--------|-------------|
| **具体的なテーマを入れる** | 「DX推進担当者向け クラウド移行セミナー」 | 「セミナーのご案内」 |
| **対象者を明記する** | 「経営層向け AI活用戦略ワークショップ」 | 「AIワークショップ」 |
| **開催形式を入れる** | 「【オンライン】採用力強化セミナー」 | 「採用セミナー」 |

## イベント詳細文のコツ

参加者が申し込みを判断するための重要な情報です。以下の要素を含めることをおすすめします。

### 記載すべき内容

| 項目 | 説明 |
|------|------|
| **イベントの概要** | どんなイベントか、何が学べるか |
| **こんな方におすすめ** | 参加対象者の具体的な職種・課題 |
| **プログラム内容** | タイムスケジュールや登壇者情報 |
| **参加メリット** | 参加することで得られる価値 |
| **参加条件・注意事項** | 対象者の条件や注意点があれば記載 |

## イベント画像のコツ

魅力的なイベント画像は、クリック率に大きく影響します。

| ポイント | 説明 |
|---------|------|
| **テキストを入れる** | イベントのテーマやキーワードを画像内に配置 |
| **視認性を高める** | 文字が読みやすい配色・フォントサイズにする |
| **ブランドカラーを統一** | 企業のブランドカラーで統一感を出す |

## ターゲット設定のコツ

適切なターゲット設定により、イベントが対象者の検索結果に表示されやすくなります。

| 設定項目 | ポイント |
|---------|---------|
| **業種** | 幅広く設定しすぎず、核となる業種を選択 |
| **従業員数** | イベント内容に合った企業規模を選択 |
| **職種** | 実際に参加してほしい職種を選択 |
| **立場** | 意思決定者向けか、実務担当者向けかを明確に |

::: tip
ターゲットを広く設定すると多くの方に表示されますが、マッチ度が下がる可能性があります。まずは核となるターゲットに絞り、必要に応じて拡大することをおすすめします。
:::

## アンケート設定のコツ

事前アンケートは、参加者の情報を把握し、イベントの質を高めるために活用できます。

| ポイント | 説明 |
|---------|------|
| **設問数は最小限に** | 最大3問まで設定可能ですが、必要な情報に絞る |
| **選択肢は具体的に** | 回答者が迷わない明確な選択肢を用意 |
| **目的を意識する** | 何のための質問か、回答をどう活用するかを考えて設計 |

## チケット設定のコツ

| ポイント | 説明 |
|---------|------|
| **わかりやすい名前** | 日程や会場を含めた名前にする（例：「4月15日 東京会場」） |
| **適切な枚数設定** | 会場のキャパシティやオンラインの上限を考慮 |
