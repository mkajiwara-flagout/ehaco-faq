# 運営へのお問い合わせ

ehaco! 運営事務局へのお問い合わせは、以下のフォームからお送りください。

## お問い合わせ方法

以下のお問い合わせフォームからお送りください。

- **お問い合わせフォーム**: [お問合せ（主催者）](https://40j2oa.share-na2.hsforms.com/2-yS1GfZ_SXCk-YxFEd4Q0w)

::: tip アクセス方法
ehaco! のすべてのページのフッター（画面最下部）に「お問合せ（主催者）」リンクが設置されています。
:::

## お問い合わせ内容の例

| 内容 | 説明 |
|------|------|
| 新規登録・承認について | 登録申請後の承認状況の確認 |
| アカウントについて | 担当者情報の変更、ログインできない場合など |
| イベント掲載について | イベント掲載に関するご相談・ご依頼 |
| 機能について | 管理画面の使い方やエラーに関するご質問 |
| その他 | 上記に当てはまらないご質問やご要望 |

## お問い合わせの前に

以下のページで解決できないかご確認ください：

- [パスワードリセット方法](/organizer/faq/password-reset) - ログインできない場合
- [主催者情報の機能](/organizer/features/organizer-info) - 公開情報の設定について
- [担当者一覧の機能](/organizer/features/staff-list) - 担当者の追加・変更について

## 参加者からの問い合わせを受け取るには

イベント参加者からの問い合わせを受け付けるには、主催者情報の「**お問い合わせURL**」を設定してください。

詳しくは [お問い合わせ方法（参加者からの問い合わせ受付）](/organizer/faq/contact) をご覧ください。
