# パスワードリセット方法

主催者アカウントのログインパスワードを忘れた場合のリセット手順をご案内します。

## リセット手順

### ステップ1：主催者ログイン画面にアクセス

ehaco! の主催者ログイン画面（ https://org.ehaco.net ）にアクセスします。

![主催者ログイン画面](/images/organizer/org-login-page.png)

### ステップ2：パスワードリセットリンクをクリック

ログインフォームの下にある「**パスワードをお忘れの方はこちら**」をクリックします。

### ステップ3：メールアドレスを入力

担当者として登録済みのメールアドレスを入力し、送信ボタンをクリックします。

### ステップ4：メールを確認

入力したメールアドレス宛にパスワードリセット用のメールが届きます。メール内のリンクをクリックして、新しいパスワードを設定してください。

::: warning 注意
- パスワードリセットメールが届かない場合は、迷惑メールフォルダをご確認ください
- 担当者登録時と異なるメールアドレスではリセットできません
- リセットリンクには有効期限があります。届いたらお早めにお手続きください
:::

## 担当者のパスワードリセット

他の担当者のパスワードリセットが必要な場合は、[担当者一覧の機能](/organizer/features/staff-list)から担当者情報を確認してください。

## それでも解決しない場合

パスワードリセットがうまくいかない場合は、[運営へのお問い合わせ](/organizer/faq/contact-support)よりご連絡ください。
