# 参加者からの問い合わせ受付

参加者からの問い合わせを受け付けるための設定方法をご案内します。

## 運営事務局へのお問い合わせ

主催者としてのご質問・ご要望は、以下のフォームからお送りください。

- **主催者向けお問い合わせフォーム**: [お問合せ（主催者）](https://40j2oa.share-na2.hsforms.com/2-yS1GfZ_SXCk-YxFEd4Q0w)

::: tip アクセス方法
ehaco! のすべてのページのフッター（画面最下部）に「お問合せ（主催者）」リンクが設置されています。
:::

## よくあるお問い合わせ内容

| 内容 | 対応 |
|------|------|
| 新規登録後の承認について | 運営事務局での確認後、登録完了メールが届きます |
| イベント掲載について | お問い合わせフォームよりご連絡ください |
| 担当者の追加・変更 | [担当者一覧の機能](/organizer/features/staff-list)をご覧ください |
| アカウント情報の変更 | [主催者情報の機能](/organizer/features/organizer-info)をご覧ください |

## 参加者からのお問い合わせを受け取るには

イベント参加者からの問い合わせを受け付けるには、主催者情報の「**お問い合わせURL**」を設定してください。設定したURLがイベント詳細ページに表示されます。

詳しくは [主催者情報の機能](/organizer/features/organizer-info) をご覧ください。
