# 運営へのお問い合わせ

ehaco! 運営事務局へのお問い合わせは、以下のフォームからお送りください。

## お問い合わせ方法

以下のお問い合わせフォームからお送りください。

- **お問い合わせフォーム**: [お問合せ（参加者）](https://40j2oa.share-na2.hsforms.com/2Ot6344zATAmokI3CPqJZ6w)

::: tip アクセス方法
ehaco! のすべてのページのフッター（画面最下部）に「お問合せ（参加者）」リンクが設置されています。
:::

## お問い合わせ内容の例

| 内容 | 説明 |
|------|------|
| アカウントについて | 登録情報の変更・削除、ログインできない場合など |
| サービスの利用方法 | 機能の使い方やエラーに関するご質問 |
| イベント掲載について | イベント掲載のご相談・ご依頼 |
| その他 | 上記に当てはまらないご質問やご要望 |

## お問い合わせの前に

以下のページで解決できないかご確認ください：

- [パスワードリセット方法](/participant/faq/password-reset) - ログインできない場合
- [アカウント登録情報の用途](/participant/faq/account-info-usage) - 個人情報の取り扱いについて
- [利用規約・プライバシーポリシー](/participant/faq/terms) - 規約に関するご質問

## イベント主催企業への問い合わせ

イベントの内容に関するご質問は、運営ではなく主催企業に直接お問い合わせください。

詳しくは [主催企業へのお問い合わせ方法](/participant/faq/contact) をご覧ください。
