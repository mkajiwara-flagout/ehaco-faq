# アカウント登録時に入力した情報はどこで使用されますか？

## 基本情報

| 項目 | 用途 |
|------|------|
| 姓名（漢字）・姓名（カナ） | イベント主催者の本人確認やehaco!運営の登録者管理において使用されます |
| パスワード | ログインの際に使用するパスワードです |
| プロフィール表示名（任意） | ヘッダーのアイコンをクリック後に出現するポップアップウィンドウに表示されます |

## 会社情報

| 項目 | 用途 |
|------|------|
| 会社名・会社URL・所在地 | イベント主催者の本人確認やehaco!運営の登録者管理において使用されます |
| 電話番号 | イベント主催者との連絡やehaco!運営の登録者管理において使用されます |

## 属性情報

| 項目 | 用途 |
|------|------|
| 業種 | イベント主催者側での情報確認に使用されます |
| 担当領域（職種） | イベント主催者側での情報確認に使用されます |
| 立場（役職） | イベント主催者側での情報確認に使用されます |
| 会社の従業員規模 | イベント主催者側での情報確認に使用されます |
| 会社の年商規模 | イベント主催者側での情報確認に使用されます |
