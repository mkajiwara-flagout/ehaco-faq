# よくあるトラブルと解決方法

ehaco! の利用中によくあるトラブルと、その解決方法をまとめています。

## ログインできない

| 原因 | 解決方法 |
|------|---------|
| パスワードを忘れた | [パスワードリセット](/participant/faq/password-reset) から再設定してください |
| メールアドレスを間違えている | 会員登録時のメールアドレスをご確認ください |
| 会員登録が完了していない | [新規会員登録](/participant/getting-started/step1-registration) から登録を行ってください |

## メールが届かない

会員登録時の確認メールやパスワードリセットメールが届かない場合：

1. **迷惑メールフォルダ**を確認してください
2. メールアドレスに**入力ミス**がないか確認してください
3. 「`@ehaco.net`」からのメールを受信できるよう、**受信許可設定**を行ってください
4. 携帯キャリアメール（docomo、au、SoftBank等）をお使いの場合、PCメールの受信がブロックされていることがあります

::: tip
上記を確認しても届かない場合は、別のメールアドレス（Gmail等）での再登録もご検討ください。
:::

## イベントに申し込めない

| 原因 | 解決方法 |
|------|---------|
| ログインしていない | ログインしてから再度お試しください |
| 募集期間が終了している | 募集期間内のイベントのみ申し込み可能です |
| 定員に達している | 満席のイベントには申し込めません |
| アンケートが未回答 | 必須アンケートにすべて回答してください |

## ページが正しく表示されない

| 原因 | 解決方法 |
|------|---------|
| ブラウザが古い | 最新版のブラウザに更新してください |
| キャッシュの問題 | ブラウザの**キャッシュをクリア**してからページを再読み込みしてください |
| JavaScript が無効 | ブラウザの設定でJavaScriptを有効にしてください |

### キャッシュのクリア方法

| ブラウザ | 操作方法 |
|---------|---------|
| **Chrome** | `Ctrl + Shift + Delete`（Mac: `Cmd + Shift + Delete`） |
| **Safari** | メニュー → 履歴 → 履歴を消去 |
| **Firefox** | `Ctrl + Shift + Delete`（Mac: `Cmd + Shift + Delete`） |
| **Edge** | `Ctrl + Shift + Delete`（Mac: `Cmd + Shift + Delete`） |

## お気に入り・ブックマークが保存されない

- ログインしていることを確認してください
- ブラウザのCookieが無効になっていないか確認してください

## それでも解決しない場合

上記の方法で解決しない場合は、[運営へのお問い合わせ](/participant/faq/contact-support) からご連絡ください。

お問い合わせの際、以下の情報をお伝えいただくとスムーズに対応できます：

- ご利用のブラウザとバージョン
- 発生した問題の詳細（エラーメッセージがあればその内容）
- 問題が発生した画面のURL
