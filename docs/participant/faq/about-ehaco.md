# ehaco! とは

## サービス概要

**ehaco!**（エハコ）は、ビジネス向けのイベント・セミナー検索サービスです。

参加者は、業界・エリア・日時などの条件でイベントを検索し、オンライン・オフラインのセミナーやイベントに申し込むことができます。

## 主な特徴

| 特徴 | 説明 |
|------|------|
| イベント検索 | キーワード・業界・エリア・日時など多彩な条件で検索できます |
| オンライン・オフライン対応 | オンライン配信のイベントにも対応しています |
| 無料で利用可能 | 参加者の方はアカウント登録・イベント検索ともに無料です |
| 主催者向け機能 | イベント主催者向けの管理ツールも提供しています |

## ご利用の流れ

### 参加者の方

1. [新規利用者登録](/participant/getting-started/step1-registration) でアカウントを作成
2. [イベント検索](/participant/getting-started/step3-search) で気になるイベントを探す
3. [イベント申込み](/participant/getting-started/step4-apply) で参加登録

### 主催者の方

1. [新規主催者登録](/organizer/getting-started/step1-registration) で主催者アカウントを作成
2. 運営事務局の承認後、ログインが可能に
3. [イベント作成](/organizer/getting-started/step5-create-event) でイベントを公開
