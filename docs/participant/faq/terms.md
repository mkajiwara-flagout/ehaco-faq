# 利用規約・プライバシーポリシーについて

ehaco! の利用規約およびプライバシーポリシーに関するご案内です。

## 利用規約

ehaco! では、参加者と主催者それぞれに利用規約が用意されています。

| 種類 | 対象 |
|------|------|
| イベント参加者向け利用規約 | イベントに参加する方 |
| イベント主催者向け利用規約 | イベントを主催する方 |

利用規約は、ehaco! のすべてのページのフッター（画面最下部）からアクセスできます。

## 個人情報の取り扱い

ehaco! では、お預かりした個人情報を適切に管理しています。詳細は、フッターの「**個人情報の取り扱いについて**」からご確認いただけます。

## 利用規約への同意が必要なタイミング

以下のタイミングで利用規約への同意が必要です：

- **新規会員登録時**: アカウント作成の際に利用規約への同意が求められます
- **イベント申込み時**: イベント申込みの際に個人情報取扱い利用規約への同意が必要です

::: warning 注意
イベント申込み時の利用規約同意はチェックボックスとなりますので、忘れずにチェックしてください。チェックがないと申込みが完了できません。
:::

## ご質問がある場合

利用規約やプライバシーポリシーに関するご質問は、[運営へのお問い合わせ](/participant/faq/contact-support)よりお送りください。
