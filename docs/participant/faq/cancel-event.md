# イベントへの参加を取り止めたい

すでに申し込みをしているイベントへの参加をキャンセルしたい場合の手順です。

## キャンセル手順

1. マイページのサイドバーにある「申込みイベント」→「募集中」にあるイベント一覧に移動します
2. キャンセルしたいイベント項目の右側にある「キャンセルする」ボタンをクリックしてください
3. 確認ダイアログが表示されるので、内容を確認して「キャンセルする」を選択します
4. キャンセルが成功すると、ご登録いただいたメールアドレスにキャンセル確認メールが届きます

::: tip
マイページへの行き方は [マイページの使い方](/participant/features/my-page) をご覧ください。
:::

## キャンセルに関する注意事項

| 項目 | 内容 |
|------|------|
| **キャンセル可能な期間** | イベントの募集期間中のみキャンセルが可能です |
| **キャンセル後の再申込** | キャンセル後でも、募集期間中であれば再度申し込みが可能です |
| **キャンセル確認メール** | キャンセル完了時に登録メールアドレス宛に確認メールが届きます |

::: warning 注意
- 募集期間が終了したイベントはキャンセルできません
- キャンセル操作は取り消せません。再度参加したい場合は改めてお申し込みください
:::

## キャンセルできない場合

キャンセルボタンが表示されない場合は、以下の原因が考えられます：

| 状況 | 対処法 |
|------|--------|
| 募集期間が終了している | 直接イベント主催者にお問い合わせください |
| イベントが終了済み | キャンセル対象外です |

イベント主催者への連絡方法は [主催企業へのお問い合わせ方法](/participant/faq/contact) をご覧ください。
