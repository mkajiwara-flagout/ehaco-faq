# 主催企業へのお問い合わせ

イベント主催企業への問い合わせ方法をご案内します。

## イベントに関するお問い合わせ

イベントの内容に関するご質問は、**イベント詳細ページ**から主催企業に直接お問い合わせいただけます。

1. イベント詳細ページを開く
2. 画面右側の主催企業情報セクションにある「**主催企業にお問合せ**」ボタンをクリック
3. 主催企業が設定したお問い合わせページに遷移します

![イベント詳細ページのサイドバー](/images/participant/event-detail-sidebar.png)
