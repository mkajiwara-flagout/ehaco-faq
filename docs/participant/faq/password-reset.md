# パスワードリセット方法

ログインパスワードを忘れた場合のリセット手順をご案内します。

## リセット手順

### ステップ1：ログイン画面にアクセス

ehaco! のログイン画面（ https://ehaco.net/login ）にアクセスします。

![ログイン画面](/images/participant/login-page.png)

### ステップ2：パスワードリセットリンクをクリック

ログインフォームの下にある「**パスワードをお忘れの方はこちら**」をクリックします。

### ステップ3：メールアドレスを入力

登録済みのメールアドレスを入力し、送信ボタンをクリックします。

### ステップ4：メールを確認

入力したメールアドレス宛にパスワードリセット用のメールが届きます。メール内のリンクをクリックして、新しいパスワードを設定してください。

::: warning 注意
- パスワードリセットメールが届かない場合は、迷惑メールフォルダをご確認ください
- 登録時と異なるメールアドレスではリセットできません
- リセットリンクには有効期限があります。届いたらお早めにお手続きください
:::

## それでも解決しない場合

パスワードリセットがうまくいかない場合は、[お問い合わせフォーム](/participant/faq/contact)よりご連絡ください。
