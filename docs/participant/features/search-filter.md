# 検索による絞り込みページの使い方

このページではより細かく絞り込み検索が行えます。

検索の絞り込みは、左のサイドバーから行うことができます。

![絞り込み検索ページ](/images/participant/search-page-full.png)

## 絞り込み方法

絞り込み方法は大きく3つあります：

1. **キーワードでの絞り込み**
2. **興味のある分野タグを用いての絞り込み**
3. **チェックボックスを用いての絞り込み**

### 使用例

「リモートワークの導入における、社内コミュニケーションツールの導入イベントを探したい、多忙なためオンライン開催がいい」場合：

| 絞り込み項目 | 設定内容 |
|------------|---------|
| キーワード | リモートワーク |
| 興味のある分野タグ | 「リモートワーク」「業務効率化」「デジタル化推進」 |
| 開催方法 | オンライン開催 |
| 申込み状態 | 受付中 |
| 対象規模 | 30-50名未満 |
| 対象職種 | 情報システム・IT、DX推進、総務・人事・法務、経営・役員 |

::: warning 注意
サイドバー下部の「絞り込みボタン」を押すと検索結果が適用されます。
:::
