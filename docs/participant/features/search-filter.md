# 検索による絞り込みページの使い方

このページではより細かく絞り込み検索が行えます。

ヘッダーの「イベントを探す」またはTOPページの検索結果から、このページにアクセスできます。

![絞り込み検索ページ](/images/participant/search-page-full.png)

## ページ上部の検索

ページ上部では、TOPページと同様に以下の絞り込みが行えます：

| 項目 | 説明 |
|------|------|
| **対象業界を選ぶ** | 関心のある業界を選択 |
| **エリアを選ぶ** | 開催地域を指定 |
| **開催日時から選ぶ** | カレンダーから希望の日程を選択 |

## サイドバーの詳細絞り込み

ページ左側のサイドバーでは、さらに細かい条件で絞り込むことができます。

### キーワード

テキストボックスに任意のキーワードを入力して検索します。

### 興味のある分野タグ

分野タグを選択してイベントを絞り込めます。タグは検索ボックスに入力して検索するか、「一覧から選択」ボタンから一覧表示して選ぶことができます。

### 開催方法

| 選択肢 | 説明 |
|--------|------|
| **オンライン開催** | Zoom等を使ったオンライン参加のイベント |
| **オフライン開催** | 会場での対面参加のイベント |

### 申込み状態

| 選択肢 | 説明 |
|--------|------|
| **受付中** | 現在申し込みを受け付けているイベント |
| **受付終了** | 申し込み受付が終了したイベント |

### 対象規模

イベントの対象となる企業規模で絞り込めます：

- 個人事業
- 2-10名未満
- 10-30名未満
- 30-50名未満
- 50-100名未満
- 100名-200名
- 200名-500名
- 500-1,000名未満
- 1,000名以上

### 対象職種

自分の職種に合ったイベントを探せます。主な選択肢：

- 経営・役員
- 経営企画・事業企画
- DX推進
- 広報・PR
- マーケティング・販売促進
- 営業・販売
- 専門コンサルタント（経営・戦略・財務）
- 専門コンサルタント（IT）
- 総務・法務・購買
- 人事・労務
- 経理・財務
- 情報システム・社内SE
- プロジェクト管理者（PM・PL）
- 各種エンジニア（ソフトウェア、ネットワーク、ハードウェア、セキュリティ、AI、データ）
- Webディレクター・Webデザイナー
- 通販・EC関連
- 広告運用・広告制作
- 商品企画・商品開発
- 研究開発・技術・生産管理
- 品質管理・カスタマーサポート
- 製造・調達
- その他

::: warning 注意
条件を設定したら、サイドバー下部の **「絞り込む」ボタン** を押すと検索結果に反映されます。ボタンを押さないと条件が適用されません。
:::

## 検索結果

検索結果は画面右側にイベントカードの一覧で表示されます。

### 並び替え

検索結果の右上にある並び替えメニューから、表示順を変更できます（新着順など）。

### イベントカードの情報

各イベントカードには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| サムネイル画像 | イベントのイメージ画像 |
| イベント名 | クリックするとイベント詳細ページに移動 |
| 開催期間 | 開始日時〜終了日時 |
| 申込締切 | チケットの申込み締切日時 |
| 定員 | 現在の申込数 / 定員数 |
| 開催方法 | オンライン / オフラインの形式と場所 |
| 主催企業名 | クリックすると主催企業ページに移動 |
| タグ | イベントに関連するカテゴリタグ |
| ブックマークボタン | お気に入りに追加（ログイン時のみ） |

### ページネーション

検索結果が多い場合は、ページ下部のページネーションで次のページに移動できます。

## 使用例

「リモートワーク関連のオンラインイベントを探したい」場合：

| 絞り込み項目 | 設定内容 |
|------------|---------|
| キーワード | リモートワーク |
| 興味のある分野タグ | 「リモートワーク」「業務効率化」 |
| 開催方法 | オンライン開催 |
| 申込み状態 | 受付中 |
| 対象職種 | 情報システム・社内SE、DX推進 |
