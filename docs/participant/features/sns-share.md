# SNS共有機能の使い方

イベント情報をSNSや他の方法で共有する方法をご案内します。

## 共有方法

イベント詳細ページの上部に、以下のSNS共有ボタンが用意されています：

| 共有先 | 説明 |
|-------|------|
| X（旧Twitter） | イベント情報をXでポストします |
| Facebook | Facebookでシェアします |
| LinkedIn | LinkedInで共有します |

## URLコピーによる共有

SNS以外にも、イベント詳細ページのURLをコピーして、メールやチャットで直接共有することもできます。

共有ボタンの横にある**リンクコピーボタン**をクリックすると、イベントページのURLがクリップボードにコピーされます。

## 共有される情報

SNSで共有すると、以下の情報が自動的に含まれます：

- イベント名
- イベントページへのリンク

::: tip 活用例
興味のあるセミナーを同僚やチームメンバーにシェアしたい場合、LinkedIn共有が便利です。
:::
