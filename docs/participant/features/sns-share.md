# SNS共有機能の使い方

イベント情報をSNSや他の方法で共有する方法をご案内します。

## 共有ボタンの場所

イベント詳細ページの下部にある **「このイベント情報を共有」** セクションに共有ボタンが用意されています。

## SNSで共有する

以下のSNSに直接共有できます：

| 共有先 | 説明 | おすすめの活用シーン |
|-------|------|-------------------|
| **X（旧Twitter）** | イベント情報をXでポストします | 幅広く情報を拡散したい場合 |
| **Facebook** | Facebookでシェアします | 友人やグループに共有したい場合 |
| **LinkedIn** | LinkedInで共有します | ビジネス関連のセミナーを同僚にシェアしたい場合 |

### 共有される情報

SNSで共有すると、以下の情報が自動的に投稿に含まれます：

- イベント名
- イベントページへのリンク

## URLコピーで共有する

SNS以外にも、イベントのURLをコピーしてメールやチャットツールで直接共有できます。

**「このページのURLをコピーする」** ボタンをクリックすると、イベントページのURLがクリップボードにコピーされます。

::: tip 活用例
- 社内のSlackやTeamsでチームメンバーに共有
- メールで取引先にセミナー情報を案内
- LINEやメッセンジャーで友人に共有
:::

## 共有時の注意点

- 共有されるのはイベントの公開情報のみです。個人情報が含まれることはありません
- 非公開イベントや募集終了後のイベントでも、URLを共有することは可能です
- イベントへの申し込みには、共有先の方もehaco! への会員登録が必要です
