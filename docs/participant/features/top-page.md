# TOPページの使い方

TOPページでは検索やおすすめなど、自分に合ったイベントを探すことができます。

![TOPページ](/images/participant/top-page-full.png)

## ヘッダー

画面上部のヘッダーには、以下の要素があります：

| 要素 | 説明 |
|------|------|
| **ehaco! ロゴ** | クリックするとTOPページに戻ります |
| **イベントを探す** | 絞り込み検索ページに移動します |
| **ehaco!（エハコ）とは** | サービス紹介ページに移動します |
| **掲載希望はこちら** | 主催者向け情報ページに移動します |
| **ユーザーアイコン**（ログイン後） | クリックするとマイページ・ログアウトのメニューが表示されます |

![ログイン後のヘッダー](/images/participant/top-page-header-loggedin.png)
*ログイン後はヘッダー右上にユーザーアイコンが表示されます*

::: tip
ユーザーアイコンをクリックすると、マイページへの移動やログアウトが行えます。
:::

## 検索機能

### キーワード検索

ページ中央の検索ボックスに、気になるキーワードを入力してイベントを検索できます。

### 業界で絞り込む

「対象業界を選ぶ」から関心のある業界を選択します。不動産、IT、金融など、さまざまな業界から選べます。

### エリアで絞り込む

「エリアを選ぶ」から開催地域を指定できます。

### 開催日時で絞り込む

カレンダーから希望の日程を選択できます。

::: info
より詳細な絞り込み条件で検索したい場合は、ヘッダーの「イベントを探す」から [絞り込み検索ページ](/participant/features/search-filter) をご利用ください。
:::

## おすすめのイベントや新着イベント

![新着・おすすめイベント](/images/participant/top-page-events.png)

TOPページには、以下のセクションでイベントが表示されます：

| セクション | 説明 |
|-----------|------|
| **あなたにおすすめのセミナー・イベント** | マイページで設定した「興味のある分野タグ」に基づいて、おすすめのイベントが表示されます |
| **新着セミナー・イベント** | 新しく公開されたイベントが表示されます。「すべてを一覧で見る」から全件表示も可能です |
| **まもなく締め切りになるセミナー・イベント** | 申込み締切が近いイベントが表示されます。気になるイベントはお早めにお申し込みください |

### イベントカードの見方

各イベントカードには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **サムネイル画像** | イベントのイメージ画像 |
| **イベント名** | クリックするとイベント詳細ページに移動します |
| **開催期間** | 開始日時〜終了日時 |
| **定員** | 参加可能な人数 |
| **開催方法** | 「オンライン（Zoom）」「オフライン（地域名）」など |
| **主催企業名** | イベントの主催企業。クリックすると主催企業ページに移動します |
| **タグ** | イベントに関連するカテゴリタグ |
| **ブックマークボタン** | クリックするとお気に入りに追加できます（ログイン時のみ） |

::: tip おすすめイベントの精度を上げるには
[マイページ](/participant/features/my-page) の「利用者情報」から **「興味のある分野タグ」** を複数設定すると、より自分に合ったイベントがおすすめされるようになります。
:::

## フッター

画面下部のフッターからは以下のリンクにアクセスできます：

| カテゴリ | リンク |
|----------|--------|
| **イベント** | 新着イベント、締切が近いイベント、イベント一覧、ログイン、マイページ |
| **開催時期から探す** | 本日、今週、今月、来月 |
| **開催方法から探す** | オンライン開催（受付中）、オフライン開催（受付中） |
| **About Us** | サービス紹介、運営会社、利用規約、プライバシーポリシー、会員登録、お問合せ |
| **SNS** | LinkedIn、X、Facebook、Instagram、YouTube |
