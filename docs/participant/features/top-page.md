# TOPページの使い方

このページでは検索やおすすめなど、自分に合ったイベントを探すことができます。

![TOPページ](/images/participant/top-page-full.png)

## 検索機能

### キーワード検索

気になるキーワードで検索できます。

### 業界で絞り込む

画面上部の「対象業界を選ぶ」から関心のある分野を選択します。不動産、ITなど、さまざまな業界から選べます。

### エリアで絞り込む

「エリアを選ぶ」から開催地域を指定できます。

### 開催日時で絞り込む

カレンダーから希望の日程を選択できます。

## おすすめのイベントや新着イベント

![新着・おすすめイベント](/images/participant/top-page-events.png)

| セクション | 説明 |
|-----------|------|
| おすすめイベント | おすすめのイベントが表示され、クリックすると詳細画面を閲覧できます |
| 新着イベント | 新着のイベントが表示され、クリックすると詳細画面を閲覧できます |
| 間もなく締切になるイベント | 間もなく締切になるイベントが表示され、クリックすると詳細画面を閲覧できます |
