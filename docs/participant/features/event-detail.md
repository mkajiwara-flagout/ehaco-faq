# イベント詳細ページの使い方

このページではイベントの詳細確認、お気に入り登録、チケット申し込み、SNS共有、主催企業へのお問い合わせなどができます。

![イベント詳細ページ](/images/participant/event-detail-top.png)

## イベント情報

イベント詳細ページでは以下の情報を確認できます：

| 項目 | 説明 |
|------|------|
| **サムネイル画像** | イベントのメインビジュアル |
| **イベント名** | イベントのタイトル |
| **開催期間** | 開始日時〜終了日時 |
| **開催方法** | 「オンライン（Zoom）」「オフライン（会場名）」など |
| **申込数** | 現在の申込み人数 |
| **定員** | イベントの定員数 |
| **詳細文** | イベントの説明、プログラム内容、登壇者情報など |

## イベントチケットと申し込み

イベントの申し込みは、ページ右側（モバイルでは下部）の **「イベントチケット」** セクションから行います。

![イベントチケット・お気に入り・主催企業情報](/images/participant/event-detail-sidebar.png)

### チケットの種類

1つのイベントに複数のチケットが用意されている場合があります（例：オンライン参加、オフライン参加など）。

各チケットには以下の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **チケット名** | チケットの名称 |
| **申込締切** | そのチケットの申込み締切日時 |
| **申込ボタン** | 「申込へ」ボタンをクリックして申し込み手続きへ進みます |

::: tip
すでに申込み済みのチケットには **「申込み済み」** と表示されます。
:::

詳しい申込み手順は [ステップ4：イベント申込み](/participant/getting-started/step4-apply) をご覧ください。

## お気に入り（ブックマーク）

イベントチケットセクションの **「お気に入り」ボタン** をクリックすると、このイベントをブックマークできます。

ブックマークしたイベントは [マイページ](/participant/features/my-page) の「お気に入り」から確認できます。

## 共有機能

### SNSで共有

「このイベント情報を共有」セクションから、イベント情報をSNSで共有できます：

| 共有先 | 説明 |
|--------|------|
| **X（旧Twitter）** | イベント情報をXでポストします |
| **Facebook** | Facebookでシェアします |
| **LinkedIn** | LinkedInで共有します |

### URLコピー

**「このページのURLをコピーする」** ボタンをクリックすると、イベントページのURLがクリップボードにコピーされます。メールやチャットで直接共有したい場合に便利です。

## 主催企業情報

ページ右側（モバイルでは下部）に主催企業の情報が表示されます：

| 項目 | 説明 |
|------|------|
| **企業名・ロゴ** | 主催企業の名称とアイコン。クリックすると主催企業の詳細ページに移動します |
| **WebサイトURL** | 主催企業のWebサイトへのリンク |
| **フォロワー数** | 主催企業のフォロワー数 |
| **お問合せボタン** | 「主催企業にお問合せ」ボタンから問い合わせができます |

![主催企業の詳細ページ](/images/participant/organizer-page-top.png)
*主催企業名をクリックすると詳細ページに移動できます*
