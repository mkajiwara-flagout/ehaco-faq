# イベント詳細ページの使い方

このページではイベントの詳細確認、お問い合わせ、イベント申し込みなどができます。

![イベント詳細ページ](/images/participant/event-detail-top.png)

## イベント情報

イベント詳細ページでは以下の情報を確認できます：

- サムネイル
- タイトル
- 開催時期
- 定員
- タグ
- 詳細文

## イベントの申し込み

イベントの申し込みは、画面右下に位置する「イベントチケット」セクションから行うことができます。

![イベントチケット・お気に入り・主催企業情報](/images/participant/event-detail-sidebar.png)

詳しくは [ステップ4：イベント申込み](/participant/getting-started/step4-apply) をご覧ください。

## 共有やお問い合わせ

### イベント情報の共有

X、Facebook、LinkedIn 経由でイベントを共有できます。また、ページのURLをコピーすることも可能です。

### 主催企業情報

主催企業のプロフィール（企業名・WebサイトURL・フォロワー数）が表示されます。「主催企業にお問合せ」ボタンから問い合わせることもできます。

![主催企業の詳細ページ](/images/participant/organizer-page-top.png)
*↑ 主催企業名をクリックすると詳細ページに移動できます*
