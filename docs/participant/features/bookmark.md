# お気に入り（ブックマーク）機能

気になるイベントをブックマークして、後から簡単に見返すことができます。

## ブックマークの方法

### イベント一覧からブックマーク

TOPページや検索結果のイベント一覧で、各イベントカードに表示されている**ブックマークボタン**（しおりアイコン）をクリックすると、お気に入りに追加できます。

### イベント詳細ページからブックマーク

イベント詳細ページの右側サイドバーにもブックマークボタンがあります。

![イベント詳細ページのサイドバー](/images/participant/event-detail-sidebar.png)

## ブックマークしたイベントの確認

ブックマークしたイベントは、**マイページ**のお気に入り一覧から確認できます。

詳しくは [マイページの使い方](/participant/features/my-page) をご覧ください。

::: tip ヒント
気になるイベントを見つけたら、まずブックマークしておくと便利です。申し込みは後からでも可能です。
:::

## ブックマークの解除

ブックマーク済みのイベントのブックマークボタンをもう一度クリックすると、お気に入りから削除されます。
