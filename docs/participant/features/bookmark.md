# お気に入り（ブックマーク）機能

気になるイベントをブックマークして、後から簡単に見返すことができます。

::: info
ブックマーク機能を利用するにはログインが必要です。
:::

## ブックマークの方法

### イベント一覧からブックマーク

TOPページや検索結果のイベント一覧で、各イベントカードに表示されている**ブックマークボタン**（しおりアイコン）をクリックすると、お気に入りに追加できます。

### イベント詳細ページからブックマーク

イベント詳細ページの「イベントチケット」セクション内にある **「お気に入り」ボタン** をクリックしてブックマークできます。

![イベント詳細ページのサイドバー](/images/participant/event-detail-sidebar.png)

## ブックマークしたイベントの確認

ブックマークしたイベントは、**マイページ**のお気に入り一覧から確認できます。

詳しくは [マイページの使い方](/participant/features/my-page) をご覧ください。

## ブックマークの解除

ブックマーク済みのイベントのブックマークボタンをもう一度クリックすると、お気に入りから削除されます。

::: tip 活用のヒント
- 気になるイベントを見つけたら、まずブックマークしておくと便利です。申し込みは後からでも可能です
- 複数のイベントを比較検討したい場合、候補をすべてブックマークしておくと見比べやすくなります
- 募集開始前のイベントをブックマークしておけば、募集開始時に見逃しを防げます
:::

## よくある質問

| 質問 | 回答 |
|------|------|
| ブックマーク数に上限はある？ | 上限はありません。気になるイベントは自由にブックマークできます |
| 募集終了したイベントはどうなる？ | ブックマークは残りますが、申し込みはできません |
| ブックマークしたイベントが削除された場合は？ | お気に入り一覧から自動的に表示されなくなります |
