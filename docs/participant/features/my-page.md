# マイページの使い方

## マイページへの行き方

1. 「ヘッダー」右端にある「アイコン」部分をクリック
2. 表示されるポップアップから「マイページ」をクリック

## マイページ内のコンテンツ

### 申込みイベント

申込み済みのイベントを管理できます。

| タブ | 内容 |
|-----|------|
| 募集中 | 申込んだイベントのうち、参加者の募集期間にあるイベント |
| 募集終了 | 申込んだイベントのうち、募集期間が終了したイベント |

::: tip
イベントのキャンセルもこちらから行えます。詳しくは [イベントへの参加を取り止めたい](/participant/faq/cancel-event) をご覧ください。
:::

### お気に入り

「ブックマーク」をつけたイベントはこちらで管理できます。

## 利用者情報

### 登録アカウント

新規登録の際に行った個人情報設定をこちらで修正できます。

また「興味のある分野タグ」をこのページで設定することで、TOPページの「あなたにおすすめのイベント」に関連したイベントが出現しやすくなります。

### プロフィール設定

ここでは以下の設定が行えます：

- アイコン画像の設定
- プロフィール名の設定

変更は主にヘッダー部分のアイコンとそのポップアップに反映されます。
