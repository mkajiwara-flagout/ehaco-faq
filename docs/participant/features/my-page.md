# マイページの使い方

ログイン後に利用できる「マイページ」では、申込みイベントの管理、お気に入りの確認、利用者情報の編集、お知らせ設定の変更が行えます。

## マイページへのアクセス方法

1. ehaco! にログインした状態で、**ヘッダー右上のユーザーアイコン**をクリック
2. 表示されるポップアップメニューから **「マイページ」** をクリック

::: tip
ログインしていない場合は、先にログインが必要です。詳しくは [ログインについて](/participant/getting-started/step2-login) をご覧ください。
:::

## マイページのメニュー構成

![マイページ 申込みイベント画面](/images/participant/mypage-events.png)

マイページの左側サイドバーに以下のメニューがあります：

| メニュー | 内容 |
|----------|------|
| **申込みイベント** | 申込み済みイベントの一覧確認・キャンセル |
| **お気に入り** | ブックマークしたイベントの一覧 |
| **利用者情報** | 個人情報・会社情報・興味タグの確認・編集 |
| **お知らせ設定** | メール通知（ニュースレター・おすすめ）の設定 |
| **ヘルプページ** | ヘルプセンターへのリンク |

## 申込みイベント

申込み済みのイベントチケットを確認・管理できます。

### 表示される情報

申込み中のイベントが一覧で表示され、各イベントには以下の情報が含まれます：

| 項目 | 説明 |
|------|------|
| **イベント名・サムネイル** | イベントの名称と画像。クリックするとイベント詳細ページに移動します |
| **開催期間** | イベントの開始日時〜終了日時 |
| **受付状態** | 「受付中」などの現在のステータス |
| **申込締切** | チケットの申込み締切日時 |
| **開催方法** | 「オンライン（Zoom）」「オフライン（会場名）」など |
| **定員** | 現在の申込数 / 定員数 |
| **チケット名** | 申し込んだチケットの種類 |
| **主催企業名** | イベントの主催企業。クリックすると主催企業ページに移動します |

### イベントのキャンセル

申込み済みイベントのキャンセルもこの画面から行えます。

1. マイページの「申込みイベント」を開く
2. キャンセルしたいイベントの **「キャンセルする」ボタン** をクリック
3. 確認画面で **「キャンセルする」** をクリック

::: warning 注意
キャンセルすると元に戻せません。再度参加したい場合は、もう一度申し込みが必要です。
:::

詳しくは [イベントへの参加を取り止めたい](/participant/faq/cancel-event) をご覧ください。

## お気に入り（ブックマーク）

ブックマークしたイベントの一覧が表示されます。

- イベントカードをクリックすると、イベント詳細ページに移動します
- お気に入りに登録したイベントがない場合は「データなし」と表示されます
- ブックマークの追加・解除方法は [お気に入り（ブックマーク）機能](/participant/features/bookmark) をご覧ください

![お気に入り一覧](/images/participant/mypage-bookmarks.png)

## 利用者情報

新規登録時に入力した個人情報や会社情報をこちらで確認・修正できます。

### 編集できる項目

#### 基本情報

| 項目 | 説明 |
|------|------|
| 氏名（漢字） | 姓・名 |
| 氏名（カナ） | セイ・メイ |
| メールアドレス | ログインに使用するメールアドレス |
| 電話番号 | 連絡先電話番号 |

#### 会社情報

| 項目 | 説明 |
|------|------|
| 会社名 | 所属する会社・組織名 |
| 会社URL | 会社のWebサイト |
| 所在地 | 会社の所在地（都道府県・住所） |

#### 属性情報

| 項目 | 説明 |
|------|------|
| 業種 | 所属する業界（金融、ITなど） |
| 担当領域（職種） | 担当している職種 |
| 立場（役職） | 現在の役職 |
| 会社の従業員規模 | 従業員数の規模 |
| 会社の年商規模 | 年間売上の規模 |
| ビジネスタイプ | BtoB / BtoC など |

### 興味のある分野タグ

このページで **「興味のある分野タグ」** を設定できます。

設定すると、TOPページの **「あなたにおすすめのセミナー・イベント」** に関連するイベントが表示されやすくなります。

::: tip おすすめ
複数のタグを設定すると、より自分に合ったイベントがおすすめされます。
:::


## お知らせ設定

ehaco! からのメール配信に関する設定を変更できます。

| 設定項目 | 説明 |
|----------|------|
| **ニュースレター配信** | ehaco! からの最新情報メールの受信ON/OFF |
| **おすすめイベント配信** | 興味タグに基づくおすすめイベントメールの受信ON/OFF |

::: tip
不要なメールが届く場合は、こちらからOFFに設定できます。
:::

## ログアウト

1. ヘッダー右上のユーザーアイコンをクリック
2. ポップアップメニューから **「ログアウト」** をクリック

ログアウトすると、ログイン画面に戻ります。
