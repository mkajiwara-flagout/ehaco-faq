# ステップ5：申し込み後の流れ

イベントへの申し込みが完了したら、以下の流れで確認・参加します。

## 1. 申し込み完了メールを確認する

申し込みが完了すると、登録メールアドレスに **申し込み完了メール** が届きます。

::: tip
メールが届かない場合は、迷惑メールフォルダもご確認ください。
:::

## 2. マイページで申込み状況を確認する

申込み済みのイベントは **マイページ** からいつでも確認できます。

1. ヘッダー右上のユーザーアイコンをクリック
2. 「マイページ」を選択
3. サイドバーの **「申込みイベント」** メニューを開く

![マイページの申込みイベント](/images/participant/mypage-events.png)

### 確認できる情報

マイページの申込みイベント画面では、以下の情報が確認できます：

| 項目 | 説明 |
|------|------|
| イベント名 | クリックするとイベント詳細ページに移動します |
| 開催期間 | イベントの開始日時〜終了日時 |
| 受付状態 | 「受付中」などの現在のステータス |
| 申込締切 | チケットの申込み締切日時 |
| 開催方法 | オンライン / オフラインの開催形式と場所 |
| 定員 | 現在の申込数と定員 |
| チケット名 | 申し込んだチケットの種類 |

::: info
マイページの詳しい使い方は [マイページの使い方](/participant/features/my-page) をご覧ください。
:::

## 3. イベント当日

### オンラインイベントの場合

- イベント詳細ページに記載された **参加URL**（Zoom等）から参加してください
- 開始時間の少し前にアクセスすることをおすすめします
- 参加URLはイベント詳細ページやメールに記載されています

### オフラインイベントの場合

- イベント詳細ページに記載された **会場住所** を確認してください
- マイページの申込みイベント画面にも開催場所（都道府県）が表示されます
- 受付時に氏名を伝えてください

## 4. イベントをキャンセルしたい場合

やむを得ず参加できなくなった場合は、マイページからキャンセルできます。

1. マイページの「申込みイベント」を開く
2. キャンセルしたいイベントの **「キャンセルする」ボタン** をクリック
3. 確認画面で **「キャンセルする」** をクリック

詳しくは [イベントへの参加を取り止めたい](/participant/faq/cancel-event) をご覧ください。

::: warning 注意
キャンセルすると元に戻せません。再度参加したい場合は、もう一度申し込みが必要です。
:::
