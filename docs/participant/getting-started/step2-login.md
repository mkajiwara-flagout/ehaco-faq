# ステップ2：ログインについて

## ログイン方法 1：ログイン画面から

新規登録後はログイン画面（ https://ehaco.net/login ）の上部から自身のアカウントに入ることができます。

チェックボックスの「ログイン状態を保持する」をクリックすると、次回からメールアドレスとパスワードの入力を省略できます。

![ログイン画面](/images/participant/login-page.png)

## ログイン方法 2：TOPページから

TOPページ（ https://ehaco.net/home ）のヘッダーからログインすることもできます。

![TOPページのヘッダー](/images/participant/top-page-header.png)
