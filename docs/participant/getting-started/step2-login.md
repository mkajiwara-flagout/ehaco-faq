# ステップ2：ログインについて

会員登録が完了したら、ehaco! にログインしてイベントの申し込みやお気に入り登録などの機能を利用できます。

## ログイン方法 1：ログイン画面から

ehaco! のログイン画面（ https://ehaco.net/login ）にアクセスし、登録したメールアドレスとパスワードを入力してログインします。

![ログイン画面](/images/participant/login-page.png)

| 入力項目 | 説明 |
|---------|------|
| **登録しているメールアドレス** | 会員登録時に設定したメールアドレス |
| **パスワード** | 会員登録時に設定したパスワード |
| **ログイン状態を保持する** | チェックを入れると、次回からメールアドレスとパスワードの入力を省略できます |

## ログイン方法 2：TOPページから

TOPページ（ https://ehaco.net/home ）のヘッダー右上にある「ログイン」ボタンからもログインできます。

![TOPページのヘッダー](/images/participant/top-page-header.png)

## ログイン後にできること

ログインすると、以下の機能が利用可能になります：

| 機能 | 説明 |
|------|------|
| **イベント申し込み** | 気になるイベントに申し込みできます |
| **お気に入り登録** | イベントをブックマークして後から確認できます |
| **マイページ** | 申込みイベントの管理、お気に入り一覧の確認ができます |
| **アンケート回答** | イベント申込時のアンケートに回答できます |

## パスワードを忘れた場合

ログイン画面の「**パスワードをお忘れの方はこちら**」リンクから、パスワードのリセットが可能です。

詳しくは [パスワードリセット方法](/participant/faq/password-reset) をご覧ください。
