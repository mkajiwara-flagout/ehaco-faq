# ステップ3：イベント検索

## どこからイベントを探す？

イベントは主に以下の2つのページから探すことができます：

- **TOPページ**（ https://ehaco.net/home ）
- **絞り込み後ページ**（ https://ehaco.net/search ）

![TOPページ](/images/participant/top-page-full.png)
*↑ TOPページ*

![絞り込みページ](/images/participant/search-page-full.png)
*↑ 絞り込みページ*

詳しい使い方は [検索による絞り込みページの使い方](/participant/features/search-filter) をご覧ください。

## 気になるイベントが見つかれば

気になるイベントが見つかれば、以下のいずれかの操作ができます：

- **ブックマークボタン**を押して後から見返せるようにする
- **イベント名**をクリックしてイベントの詳細を見る
