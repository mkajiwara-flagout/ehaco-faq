# ステップ4：イベント申込み

## イベントの申し込み

イベントの参加申込みは「イベント詳細ページ」から可能です。

イベント詳細ページへの入り方は [ステップ3：イベント検索](/participant/getting-started/step3-search) をご確認ください。

![イベント詳細ページ](/images/participant/event-detail-info.png)

## イベント登録の手順

### ステップ1：イベントチケットの選択

画面右下に「イベントチケット」という項目があります。いずれかを選択してイベントの登録を進めてください。

![イベントチケットセクション](/images/participant/event-ticket-area.png)

**例：**

| 参加方法 | 選択するチケット |
|---------|---------------|
| オンラインで参加したい場合 | 下2つのチケットのいずれか |
| オフラインで参加したい場合 | 一番上のチケット |

### ステップ2：イベント参加者情報の入力

イベントチケット選択後、以下の入力が求められます：

- イベント申込者情報入力
- イベントの事前アンケート回答
- 個人情報取扱い利用規約への同意

::: warning 注意
利用規約の同意はチェックボックスとなりますので、忘れずにご記入ください。
:::

記入後「申込みを確定ボタン」から申し込みを完了できます。

## イベント申込み成功の確認

上記の手順が正しく完了していれば、ご登録いただいたメールアドレスに確認メールが届きます。
