# ステップ1：新規利用者登録

## 1. 新規登録画面へのアクセス

ehaco! の利用者ログイン画面（ https://ehaco.net/login ）にアクセスしてください。

会員登録用の画面が表示されるので、登録したいメールアドレスをボックス内に記入して、送信ボタンを押してください。

![ログイン画面](/images/participant/login-page.png)

## 2. メールアドレスの登録

1. 登録するメールアドレスを入力
2. 「送信」ボタンをクリック
3. 入力したメールアドレス宛に専用URLが届きます

![会員登録画面](/images/participant/registration-page.png)

::: tip
メールが届かない場合は、迷惑メールフォルダもご確認ください。
:::

## 4. 利用規約の確認

以下の内容を確認してください：

- 利用規約
- プライバシーポリシー

内容を確認後、「利用規約に同意して登録情報を送信する」をクリックしてください。

## 5. 登録完了

登録完了後、以下のサービスがご利用いただけます：

- セミナー・イベントへの申込み
- イベントのお気に入り登録
